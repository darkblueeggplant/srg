Посмотреть ip-адреса машины и таблицу маршрутизации через консоль.
ip a sh
ip route show
Вывести ip-адреса в текстовый файл.
hostname -I > ip_addresses.txt 
Отфильтровать конкретный ip-адрес из текстового файла.
172.25.141.143 
Посмотреть занятое пространство на дисках. 
du -Th
Посмотреть размеры всех файлов в домашнем каталоге и подкаталогах.
du -ah ~
Посмотреть порты, которые слушает сервер. (т. е. список открытых портов, по которым сервер предоставляет соединение).
netstat -tulpn или ss -tulpn
Составить команду для вывода в файл 1000 пронумерованных строк в цикле вида «test n», где n - номер строки.
for i in {1..1000}; do echo "test $i"; done >> output.txt
Показать общие логи с момента включения системы. (Обычно, начинаются с объявления версии ядра: Linux version, второй строчкой — Command Line)
dmesg
Создать задачу на добавление текущего системного времени в текстовый файл каждые 5 минут. Показать результат работы через полчаса.
*/5 * * * * date >> /home/user/time_log.txt
